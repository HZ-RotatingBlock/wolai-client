# wolai-client

使用 electron 创建 wolai 桌面客户端


## 环境要求

- macOS 10.9+ / Windows / Linux
- [Node.js](https://nodejs.org/zh-cn/download/) `>= 10`


下载并解压本项目，进入项目文件夹下

安装依赖

```bash
npm run install
```

构建客户端

```bash
npm run build
```

 🥳需要自定义css可以修改src/customjs.js
